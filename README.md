# 🧠 Brain Tumor Detection & Analysis (DTCWT)

This project presents a Brain Tumor Detection web application using
Dual-Tree Complex Wavelet Transform (DTCWT) based feature extraction
and classical machine learning classifiers.

## 🚀 Features
- CLAHE enhancement
- DTCWT feature extraction
- Entropy analysis
- ROC curve visualization
- Tumor segmentation (compact 3-row layout)
- Flask-based web interface
